Plant Disease
Detection System

In this project, On the basics of their leaves, our model detects whether the plant has any disease or not. And if the plant has any disease, it recommends a medicine for that. For implementation, we make a CNN for detecting which contains five convolution and max pooling layers after that it has two fully connected layers. We reach an accuracy of 95.50% for classifying the leaves.

Object Measurement using OpenCV

The entire project is built to measure the dimensions of any object like a Book, USB, Phone, etc. by using OpenCV and Aruco Marker in a very simple and easy manner.The project is divided into two parts: First, we detect the object using several CNNs and after that, we use canny edge detection techniques by OpenCV for measuring the dimension of an object.
